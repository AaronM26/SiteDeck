# Analysis Website

This project provides a web-based interface to analyze and visualize various aspects of a website's design, performance, and user engagement. It comes with a responsive and themeable frontend that includes a dynamically generated heatmap, scrollable descriptive analysis cards, and a navigation bar with dropdown menus for additional context and navigation.
